# Hotel Room Booking Website

A responsive hotel room booking landing page that displays room categories with different prices, live sample availability, booking slots, add-ons, and an interactive booking estimate form.

## Room Prices

- Super Deluxe Room: ₹4,500 per night
- Deluxe Room: ₹3,500 per night
- Normal AC Room: ₹2,500 per night
- Non AC Room: ₹1,500 per night

## Booking Features

- Live sample inventory for each room category
- Arrival slot selection for standard, afternoon, evening, and late check-ins
- Guest capacity and room availability validation
- Optional breakfast and airport pickup add-ons
- Tax and service-fee breakdown in the booking estimate

## How to Run

Open `index.html` in a web browser, or serve the folder with any static file server.
