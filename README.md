# Hotel Room Booking Website

A responsive hotel room booking landing page that displays room categories with different prices and includes an interactive booking estimate form.

## Room Prices

- Super Deluxe Room: ₹4,500 per night
- Deluxe Room: ₹3,500 per night
- Normal AC Room: ₹2,500 per night
- Non AC Room: ₹1,500 per night

## How to Run

Open `index.html` in a web browser, or serve the folder with any static file server.
